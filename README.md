# Running/Building

To run and build the application you need to do the following:

[Install Trunk](https://trunkrs.dev/#install). It's used for building the frontend.



To build the example:
```bash
cd examples/full
trunk build
```

To build and serve the example website:
```bash
cd examples/full
trunk serve
```
